# Vintage Parts (Partsu Misete)

A database of vintage electronic parts.

For convenience, install the Command Daemon to build and run your software each time
you save a file.

```bash
go get github.com/githubnemo/CompileDaemon
go install github.com/githubnemo/CompileDaemon
CompileDaemon -command="./vintage-parts
```

You will require a .env file on your system e.g.

```ini
DATABASE_URL="postgresql://[user[:password]@][host][:port][/dbname]"
```

The project relies on code generated by OAPI. You will need to regenerate that code if
you change the API specification.

```bash
oapi-codegen --config=api/cfg.yaml postman/specs/openapi.yaml
```

You may wish to run the server from the command line:

```bash
go run main.go
```

You can test the API in the browser directly at the [default address](http://localhost:8080) e.g.

[List Manufacturers](http://localhost:8080/manufacturers)

Starting the Vite server:

```bash
npm run dev
```
